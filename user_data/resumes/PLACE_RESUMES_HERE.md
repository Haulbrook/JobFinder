# Place Your Resumes Here

Add your resume files to this directory.

## Recommended Files

- `resume_main.pdf` - Your primary resume (PDF format for applications)
- `resume.docx` - Editable Word version (for ATS systems)
- `resume.txt` - Plain text version (for easy parsing)

## Tips

1. **File Naming**: Use descriptive names
   - `resume_main.pdf`
   - `resume_frontend.pdf`
   - `resume_senior_engineer.pdf`

2. **Keep Updated**: Update your resume as you gain experience

3. **Multiple Versions**: Create tailored versions for different roles:
   - Technical focus
   - Management focus
   - Industry-specific

## Using in JobFinder

Reference your resume when creating your profile:

```
"Create my job search profile:
- Resume path: /Users/thehaulbrooks/Desktop/JOB FINDER/user_data/resumes/resume_main.pdf
- ..."
```

Or use the full path from your system:
```
- Resume path: ~/Desktop/JOB FINDER/user_data/resumes/resume_main.pdf
```
